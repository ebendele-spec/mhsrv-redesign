### What the numbers say about MHSRV today
1. **You already earn traffic the right way.** Organic is your #1 channel (53%) and organic/paid keyword split is 61/39 — the inverse of Camping World, which *buys* 40%+ of its visits. Your traffic quality is a strength to compound, not a problem to fix.
2. **The gap is keyword breadth, not authority per keyword.** 2.3K ranking keywords vs Camping World's 24.9K. Your top organic keywords are telling: *"rv ace," "gm class c motorhome," "2019 coachmen orion 21rs for sale," "2025 thor gemini 22mt brochure."* Long-tail unit/model/brochure searches — exactly the intent the new site's page architecture is built to capture at 10× scale.
3. **Backlinks are your biggest measurable weakness.** Similarweb shows **one** referring website. Bish's and General RV earn links from manufacturer pages, tow-guide citations, and press. This is the cheapest big lever you have.
4. **Growth is already positive** (+8.4% MoM, global rank improved 26K places in 3 months) — the goal is to bend that curve, not rescue it.

## 3. What the organic winners actually do

**General RV (best organic position in the peer set, cat. #188):**
- Rock-solid *category × location* architecture: `/rvs-for-sale/class-a` × 40+ store pages, each indexable and interlinked.
- "Nation's Largest RV Dealer" positioning repeated sitewide = entity reinforcement for brand + superlative queries.
- Deep brand/manufacturer landing pages that intercept "jayco dealer near me"-type demand.

**Bish's RV (the content machine):**
- Clean, crawlable **type taxonomy** (`/product/motor-home-class-a-diesel`, `/product/toy-hauler-fifth-wheel`… every niche gets a page — even teardrops and overland rigs).
- A WordPress **/blog/ in a subfolder** (not subdomain) answering every buyer question — the classic "They Ask, You Answer" playbook; plus utility assets (`/tow-guides`, `/delivery-estimator`) that earn links and repeat visits.
- Result: a smaller dealer that punches far above its rooftop count in organic.

**Camping World:**
- Wins on brute force: paid search + 24.9K keywords + a parts/accessories e-commerce catalog that ranks for millions of product queries and feeds the RV funnel. Not your fight to pick — but note their organic share of voice is *beatable* in motorized categories.

## 4. MHSRV's strategy: own motorized depth

Your unfair advantages nobody in that table can copy: **1,200+ motorhomes at one location, 4,571 owner reviews, the #1-in-the-world claim, and a YouTube channel with 600+ unit walkthroughs.** The new site should convert each into a ranking asset.

### Already built into the new site ✅
| Asset | SEO function |
|---|---|
| 4,571 static review pages w/ Review schema + full text | Unique content moat; long-tail "brand/model review" queries; E-E-A-T signals no competitor can fake |
| `redirects-301.csv` for the old review domain | Transfers years of review-domain equity into the main site |
| 1,224 static VDPs, unique formatted descriptions, all photos, docs, videos | Long-tail unit queries (already your best keywords) |
| **NEW today:** Product/Offer schema + meta descriptions on all 1,224 VDPs | Rich results (price, availability) in SERPs → CTR lift |
| **NEW today:** XML sitemaps (main + reviews) + robots.txt | Full-corpus indexation; 5,800 URLs discoverable day one |
| Review + AutoDealer schema, 4.6★ Google feed | Star-rating rich snippets; local entity signals |
| Static HTML pages, no client-rendering for content | Core Web Vitals headroom Camping World can't match |
| 625 embedded YouTube walkthroughs | Video schema opportunity + dwell time |

### Build next (highest impact first)
1. **Category × intent landing pages** *(the single biggest gap vs General RV/Bish's).* Static, crawlable pages for every type ("New Diesel Pushers for Sale"), type × brand ("Thor Class C Motorhomes"), type × feature ("Bunkhouse Class C RVs," "4x4 Class B Vans"), and type × price ("Class A under $150K"). The smart-search engine already computes these slices — render the top ~150 combinations as static pages with intro copy, FAQs, and live inventory. This is where the 2.3K→10K+ keyword jump lives.
2. **Model-family hub pages.** One evergreen page per family you dominate (Palazzo GT, Tellaro, Entrada, Patriot…) — specs, floorplans, video, every in-stock unit, owner reviews filtered to that model, price history. These become *the* destination Google prefers over manufacturer pages because yours have inventory + reviews + price.
3. **Answer-engine content hub** (`/guides/` subfolder, Bish's-style): "Class A vs Class C," "What does a diesel pusher cost to own," "Sprinter vs Transit vs ProMaster camper vans." Repurpose your YouTube scripts — 600+ videos are pre-written articles. This also feeds **AI-search visibility** (ChatGPT/Gemini citations), which Similarweb now flags as the most underutilized channel across the sector.
4. **Backlink program.** Target: manufacturer "find a dealer" pages (you sell 20+ brands — each should link), Blue Compass network cross-links, press around the #1-in-the-world Stats Surveys result each year, tow-guide/brochure-library citations, local Texas/DFW media. Going from 1 referring domain to even 50 quality domains changes your ceiling.
5. **Location entity pages** for Alvarado, Montgomery, Palm Desert — full LocalBusiness schema, hours, staff, reviews per location — to win "rv dealer near me / texas / alabama / california" map-pack + organic.
6. **Video schema on the 625 embed pages** + upload transcripts, so unit walkthroughs earn video rich results.

### Guardrails (performance = ranking)
- Keep pages static-first as built; lazy-load everything below the fold (already done for galleries).
- Split `reviews-data.js` (1.6MB) into paginated JSON chunks before launch so the hub's LCP stays green.
- Serve images via the CDN with width params (already done) and add `fetchpriority="high"` to each page's hero image.
- When the live feed lands, regenerate sitemaps nightly and ping Google — freshness on 1,200 in-stock URLs is a real signal for "for sale" queries.
